# Auth Module Console

Модуль аутентификации для консольных приложений на C# с валидацией логина и пароля.

<img width="338" height="166" alt="Снимок экрана 2026-03-13 200658" src="https://github.com/user-attachments/assets/519c2eb0-9dd7-4206-b27b-1daecc63ae8e" />

## Функциональность

- Ввод логина с клавиатуры
- Проверка длины логина (минимум 5, максимум 12 символов)
- Проверка соответствия логина
- Запрос пароля при успешном логине
- Проверка пароля
- Информативные сообщения об ошибках

## Пример работы

Успешная авторизация:
Введите логин:
ArmfeterLex
Введите пароль:
Arm
Добро пожаловать!

Ошибка валидации:
Введите логин:
Alex
Логин очень короткий!
Вход воспрещён!

## Примечание

Логин и пароль заданы в коде: "ArmfeterLex" / "Arm".
